pweb 주차할인 계산기

pweb.kr 웹 할인등록 화면에서 “0원까지 할인권을 얼마나 더 적용해야 하는지”를 자동으로 계산해 주는 보조 패널

1. 설치 — PC (크롬 등)

🅿️ pweb 주차할인 계산기
위 파란 버튼을 마우스로 끌어서 브라우저 즐겨찾기 바(북마크 바)에 놓으면 설치 끝입니다.
즐겨찾기 바가 안 보이면 크롬에서 Ctrl+Shift+B. 끌어다 놓기가 안 되면 아래 휴대폰 방식(수동 등록)과 동일하게 하면 됩니다.

2. 설치 — 휴대폰

휴대폰은 끌어다 놓기가 안 되므로, 아래 코드를 복사해서 북마크 URL 칸에 붙여넣습니다. 북마크 이름은 짧게 할인계산을 추천합니다(주소창에 치기 편함).

Android · Chrome
  1. 아무 페이지에서 ⋮ 메뉴 → ☆(북마크 저장)
  2. ⋮ 메뉴 → 북마크 → 방금 저장한 북마크 편집 → 이름을 할인계산으로, URL을 복사한 코드로 교체
  3. 실행: 할인등록 페이지를 띄운 상태에서 주소창에 “할인계산” 입력 → 목록에 뜨는 북마크를 탭
크롬은 북마크 목록에서 바로 탭하면 실행되지 않을 수 있어, 주소창 방식이 확실합니다.
iPhone · Safari
  1. 아무 페이지에서 공유 → 즐겨찾기에 추가, 이름 할인계산
  2. 책갈피(📖) → 즐겨찾기 편집 → 방금 항목의 URL을 복사한 코드로 교체
  3. 실행: 할인등록 페이지에서 책갈피(📖) → 할인계산 탭
삼성 인터넷
  1. 아무 페이지에서 메뉴 → 페이지 추가 → 북마크
  2. 북마크 목록에서 방금 항목 편집 → 이름 할인계산, URL을 복사한 코드로 교체
  3. 실행: 할인등록 페이지를 띄운 채 북마크 목록에서 할인계산 탭. 안 되면 주소창에 “할인계산”을 입력해 선택
이 설치 파일(install.html)을 카톡·메일로 받아 휴대폰에서 열어도 위 복사 버튼을 그대로 쓸 수 있습니다.

3. 사용법

  1. pweb.kr 할인등록 페이지(예: a23021.pweb.kr — 주소 앞부분은 건물 번호라 건물마다 다르며, 어느 건물이든 동작합니다)에 로그인한 상태에서
  2. 차량번호를 조회하고 목록에서 차량을 선택합니다 (원래 하던 대로)
  3. 북마크의 할인계산을 실행
  4. 패널이 뜨고(PC는 오른쪽 위 카드, 휴대폰은 화면 아래 시트) 선택한 차량 기준으로 1초마다 자동 갱신됩니다
다시 실행하면 패널이 닫힙니다. PC에서는 패널 제목줄을 끌어 위치를 옮길 수 있습니다.

4. 무엇을 보여주나

헤더 색으로 상태 표시 — 초록 ✓ 0원 완료 / 빨강 ⚠ 적용 필요 / 회색 대기.

✓ 0원 완료 — 남은 여유가 “1시간 20분 남음 · 18:42까지 0원”처럼 크게 표시되고 1초마다 줄어듭니다.

⚠ 적용 필요 — 부족한 시간과 함께 적용할 주차권이 카드 목록으로 나옵니다. 실제로 주차권을 적용하면 해당 카드에 ✓ 적용됨이 붙고, 남은 부족분은 다시 계산됩니다 (추천과 다른 주차권을 써도 자동으로 맞춰 갑니다).

⛔ 사용 금지 — 건물마다 사이트 구조가 다를 수 있어, 도구가 아는 구조와 다른 점이 감지되면 모든 기능을 스스로 중단하고 사용 금지 안내를 표시합니다. 이 화면이 뜨면 계산 결과를 믿지 말고 강주상 (tsusai@msn.com) 에게 문의하세요.

5. 계산 규칙

이 화면은 요금(원)을 계산하지 않고 시간만 다루므로, 도우미도 시간으로 계산합니다.

항목값
기본 무료시간30분
무료2시간(평일)120분 · 0원 · 1회
30분권 / 1시간권30분 500원 / 60분 1,000원
4시간권 / 24시간권240분 4,000원 / 1,440분 15,000원

커버 시간 = 기본무료 30분 + 적용된 할인 시간 합계. 커버가 경과시간보다 크면 0원, 모자라면 무료2시간을 먼저 쓰고 남는 만큼을 최소 비용 조합으로 채웁니다. 항상 경과시간을 넘겨 덮도록 안전하게 올립니다.

주차장 규칙(기본 무료시간·할인권 종류)이 바뀌면 bookmarklet-source.js 맨 위의 BASE_FREE·TICKETS 값만 고치면 됩니다.
