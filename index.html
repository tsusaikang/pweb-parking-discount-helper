pweb 주차할인 계산기

pweb.kr 웹 할인등록 화면에서 “0원까지 할인권을 얼마나 더 적용해야 하는지”를 자동으로 계산해 주는 보조 패널

1. 북마클릿이 뭔가요?

작은 프로그램이 담긴 북마크(즐겨찾기)입니다. 보통 북마크는 누르면 다른 페이지로 이동하지만, 북마클릿은 이동하지 않고 지금 보고 있는 페이지 위에서 잠깐 실행됩니다. 앱이나 프로그램을 설치하는 게 아니라 북마크 한 줄이 추가되는 것뿐이라, 지우면 흔적 없이 사라집니다.

2. 설치 전 — 예전 버전부터 지우세요

예전에 "주차 0원 도우미", "주차0원" 같은 이름으로 설치한 적이 있다면 먼저 삭제하세요. 남겨두면 어느 것이 최신인지 헷갈리고, 옛것을 누르면 잘못된 안내가 나올 수 있습니다.

  • PC: 북마크 바에서 해당 북마크 우클릭 → 삭제
  • 휴대폰: 브라우저 메뉴 → 북마크 → 해당 항목 길게 눌러 삭제

3. 설치 — PC (크롬 등)

아래 파란 버튼을 마우스 왼쪽 버튼으로 누른 채 그대로 화면 위쪽 북마크 바(주소창 바로 아래 가로줄)까지 끌고 가서 놓으세요. 북마크 바에 🅿️할인계산 항목이 새로 생기면 그게 설치 완료입니다 — 클릭·회원가입·설치 프로그램 전부 없습니다.

🅿️할인계산
북마크 바가 안 보이면 크롬에서 Ctrl+Shift+B 를 누르세요.
끌어다 놓기가 잘 안 되면 아래 휴대폰 방식(수동 등록)과 동일하게 하면 됩니다.
북마크 바 자리를 아끼려면: 설치된 북마크 우클릭 → 수정에서 이름을 전부 지우면 🅿️ 아이콘만 남습니다.
확장 프로그램 설치가 가능한 PC(개인 PC 등)라면, 아래 휴대폰 섹션의 방법 A(자동 실행)를 PC 크롬·엣지에서도 그대로 쓸 수 있습니다 — 클릭조차 없이 할인등록 화면을 열면 계산기가 떠 있습니다.

4. 설치 — 휴대폰

방법이 두 가지입니다. 방법 A(자동 실행)가 훨씬 편합니다 — 설치도 탭 몇 번이고, 이후에는 할인등록 화면을 열기만 하면 계산기가 저절로 떠 있습니다 (북마크도, 주소창 입력도 없음). Edge·Firefox·iPhone 이면 A를, 크롬(안드로이드)만 쓴다면 크롬은 확장을 지원하지 않으므로 B(북마클릿)를 따라 하세요.

A자동 실행 방식 (추천 — Edge · Firefox · iPhone Safari)
  1. 브라우저에 Tampermonkey 확장을 먼저 설치합니다 — 안드로이드용 Edge: 하단 ☰ 메뉴 → 확장에서 설치 / 안드로이드용 Firefox: 메뉴 → 부가 기능에서 설치
  2. 아래 초록 버튼을 탭 합니다. 그러면 Tampermonkey 의 설치 확인 화면이 새로 나타나는데(스크립트 이름 "pweb 주차할인 계산기"가 보임), 거기서 설치 버튼을 탭
  3. 끝. 이제 할인등록 화면을 열면 계산기가 자동으로 떠 있습니다
📥 자동 실행 스크립트 설치
iPhone: App Store 에서 무료 앱 “Userscripts” 설치 → 설정 → Safari → 확장 프로그램에서 켜기 → Safari 로 이 페이지를 열어 위 초록 버튼 탭.
PC 크롬·엣지 필수 설정: Tampermonkey 는 설치만 하면 조용히 동작하지 않습니다(브라우저 보안 규격). 주소창에 edge://extensions(크롬은 chrome://extensions) 입력 → 내 확장 목록에서 Tampermonkey 클릭(세부 정보) → “사용자 스크립트 허용” 토글 ON → 할인등록 페이지 새로고침. 이 항목이 없는 구버전 브라우저는 확장 페이지 구석의 “개발자 모드”를 켜세요. Tampermonkey 아이콘에 파란 안내 배너가 떠 있다면 그걸 눌러도 같은 설정으로 연결됩니다.
스크립트는 할인등록 화면에서만 동작하며, 다른 페이지에서는 아무것도 하지 않습니다.
B북마클릿 방식 (크롬 등)

① 코드 복사 → ② 북마크에 붙여넣기 두 단계입니다. (안드로이드 크롬처럼 확장을 지원하지 않는 브라우저용)

1먼저, 코드를 복사하세요

아래 버튼 한 번이면 됩니다. 복사만 되면 절반은 끝났습니다.

2그다음, 복사한 코드를 북마크에 붙여넣으세요

휴대폰 브라우저에는 "빈 북마크 새로 만들기"가 없습니다. 그래서 순서가 이렇습니다:
ⓐ 아무 페이지나 일단 북마크로 저장 → ⓑ 방금 저장된 그 북마크를 편집해서, 이름을 할인계산으로 바꾸고 URL을 지운 자리에 복사한 코드를 붙여넣기.
새로 입력하는 게 아니라 저장해 둔 것을 고치는 방식입니다.

5. 사용법 — 딱 세 단계

  1. 주차 할인 사이트(pweb.kr)에 로그인해서, 평소 하던 대로 차량번호 조회 → 차량 선택
  2. 북마크 바의 🅿️할인계산 클릭 (휴대폰 크롬은 주소창에 “할인계산” 입력 · 자동 실행 방식이면 이 단계 자체가 없습니다)
  3. 화면에 뜨는 패널이 어떤 주차권을 몇 장 적용하면 되는지 알려줍니다 — 그대로 적용하면 끝
패널은 1초마다 스스로 갱신됩니다. 한 번 더 클릭하면 닫히고, PC에서는 제목줄을 끌어 위치를 옮길 수 있습니다.

8. 업데이트 — 신경 쓸 것 없음

북마크에 들어가는 건 몇 줄짜리 로더뿐이고, 실제 계산기는 클릭할 때마다 이 사이트에서 항상 최신 버전으로 받아 실행됩니다. 즉 한 번 설치하면 업데이트를 위해 다시 손댈 일이 없습니다.

예전(코드 내장형) 버전을 쓰고 계셨다면 패널에 주황색 업데이트 안내가 뜹니다 — 그 경우에만 이 페이지에서 한 번 재설치하면 됩니다.

9. 개인정보

이 도구는 이미 열려 있는(로그인된) 페이지가 스스로 불러온 데이터를 읽어 화면에 표시합니다. (모바일 할인등록 화면에서는 적용 내역 확인을 위해 같은 사이트의 조회 기능을 읽기 전용으로 사용합니다.) 어떤 값도 외부로 전송하거나 저장하지 않습니다. 계정 정보도 다루지 않습니다. 외부와의 통신은 클릭 시 이 페이지에서 계산기 코드를 받아오는 것 하나뿐이며, 이때도 차량번호 등 어떤 데이터도 함께 보내지 않습니다.
