pweb 주차할인 계산기

pweb.kr 웹 할인등록 화면에서 “0원까지 할인권을 얼마나 더 적용해야 하는지”를 자동으로 계산해 주는 보조 패널

1. 이게 뭔가요?

pweb.kr 할인등록 화면 위에 "0원까지 할인권을 몇 장 더 적용해야 하는지"를 계산해 주는 패널을 띄우는 보조 도구입니다. 설치 방법이 두 가지인데, 방법 A(자동 실행)를 추천합니다:

  • 방법 A — 자동 실행 (추천, PC·휴대폰 공통): 브라우저 확장(Tampermonkey)에 스크립트를 한 번 등록하면, 이후 할인등록 화면을 열 때마다 계산기가 저절로 떠 있습니다. 누를 것도, 기억할 것도 없습니다.
  • 방법 B — 북마클릿 (차선): 확장 프로그램을 설치할 수 없는 환경용입니다 (확장 설치가 막힌 회사 PC, 확장을 지원하지 않는 안드로이드 크롬 등). 북마크를 눌러서 실행합니다.

2. 설치 전 — 예전 버전 정리

예전에 "주차 0원 도우미", "주차0원" 같은 이름으로 설치한 적이 있다면 먼저 삭제하세요. 남겨두면 어느 것이 최신인지 헷갈리고, 옛것을 누르면 잘못된 안내가 나올 수 있습니다. 기존 북마클릿("할인계산")을 쓰다가 방법 A로 갈아타는 경우에도, 헷갈리지 않게 북마크는 지우는 걸 추천합니다.

  • PC: 북마크 바에서 해당 북마크 우클릭 → 삭제
  • 휴대폰: 브라우저 메뉴 → 북마크 → 해당 항목 길게 눌러 삭제

3. 방법 A — 자동 실행 설치 (추천 · PC·휴대폰 공통)

A자동 실행 방식

기기별로 Tampermonkey(또는 Userscripts) 확장을 설치한 뒤, 아래 초록 버튼으로 스크립트를 등록하면 끝입니다.

💻 PC — 크롬·엣지 아래 3단계를 순서대로 하세요.

1Tampermonkey 확장 설치

쓰는 브라우저의 스토어에서 Tampermonkey(무료)를 설치합니다. 아래 버튼을 누르면 설치 페이지로 바로 이동합니다.

🟦 Edge 스토어에서 설치 🟨 Chrome 웹스토어에서 설치

열린 페이지에서 “설치”/“가져오기”를 눌러 확장을 추가하세요. (edge://extensions 목록에는 설치한 뒤에야 Tampermonkey가 나타납니다.)

2확장 설정 — “사용자 스크립트 허용” 켜기 (필수)
  1. 주소창에 edge://extensions(크롬은 chrome://extensions) 입력
  2. 내 확장에서 Tampermonkey → 세부 정보 클릭
  3. “사용자 스크립트 허용” 토글을 ON (항목이 없는 구버전은 “개발자 모드”를 켜세요)
⚠️ 이걸 안 켜면 설치돼도 아무 반응 없이 조용히 동작하지 않습니다(브라우저 보안 규격). 토글을 켠 뒤에는 Edge를 완전히 종료했다가 다시 켜세요 — 그래야 적용됩니다(창 새로고침만으론 부족).
3스크립트 설치 — 아래 초록 버튼

이 페이지 맨 아래의 초록 버튼을 누르면 Tampermonkey 설치 확인 화면(스크립트 이름 "pweb 주차할인 계산기")이 뜹니다. 거기서 설치를 누르면 끝입니다.

📱 Android — Edge·Firefox (안드로이드 크롬은 확장 미지원 → 방법 B)

  1. 안드로이드용 Edge: 하단 ☰ 메뉴 → 확장에서 Tampermonkey 설치 / 안드로이드용 Firefox: 메뉴 → 부가 기능에서 설치
  2. 아래 초록 버튼을 탭 → Tampermonkey 의 설치 확인 화면에서 설치 탭

📱 iPhone — Safari

  1. App Store 에서 무료 앱 “Userscripts” 설치 → 설정 → Safari → 확장 프로그램에서 켜기
  2. Safari 로 이 페이지를 열어 아래 초록 버튼 탭 → 설치
📥 자동 실행 스크립트 설치
설치 후 별도 실행 없이, 할인등록 화면을 열면 계산기가 자동으로 떠 있습니다. 스크립트는 할인등록 화면에서만 동작하며 다른 페이지에서는 아무것도 하지 않습니다.
4방법 B — 북마클릿 설치확장이 안 되는 환경용 (회사 PC·안드로이드 크롬)

북마클릿은 "작은 프로그램이 담긴 북마크(즐겨찾기)"입니다. 누르면 페이지 이동 대신 지금 보고 있는 페이지 위에서 잠깐 실행됩니다. 앱 설치가 아니라 북마크 한 줄이 추가되는 것뿐이라 지우면 흔적 없이 사라집니다. 자동 실행이 아니므로 쓸 때마다 북마크를 눌러야 한다는 점만 방법 A와 다릅니다.

💻 PC — 즐겨찾기 바에 끌어다 놓기

아래 파란 버튼을 마우스 왼쪽 버튼으로 누른 채 그대로 화면 위쪽 북마크 바(주소창 바로 아래 가로줄)까지 끌고 가서 놓으세요. 북마크 바에 🅿️할인계산 항목이 새로 생기면 그게 설치 완료입니다 — 클릭·회원가입·설치 프로그램 전부 없습니다.

8. 업데이트 — 신경 쓸 것 없음

어느 방법이든 실제 계산기는 실행할 때마다 이 사이트에서 항상 최신 버전으로 받아 실행됩니다. 방법 A의 등록 스크립트 자체도 Tampermonkey 가 자동으로 갱신합니다. 즉 한 번 설치하면 업데이트 때문에 다시 손댈 일이 없습니다.

아주 예전(코드 내장형) 북마클릿을 쓰고 계셨다면 패널에 주황색 업데이트 안내가 뜹니다 — 그 경우에만 이 페이지에서 한 번 재설치하면 됩니다.

9. 개인정보

이 도구는 이미 열려 있는(로그인된) 페이지가 스스로 불러온 데이터를 읽어 화면에 표시합니다. (모바일 할인등록 화면에서는 적용 내역 확인을 위해 같은 사이트의 조회 기능을 읽기 전용으로 사용합니다.) 어떤 값도 외부로 전송하거나 저장하지 않습니다. 계정 정보도 다루지 않습니다. 외부와의 통신은 클릭 시 이 페이지에서 계산기 코드를 받아오는 것 하나뿐이며, 이때도 차량번호 등 어떤 데이터도 함께 보내지 않습니다.

10. 버전 이력

버전은 대.중.소 세 자리로 매깁니다. 대(첫 자리) = 쓰는 방식이 근본적으로 바뀌는 큰 변화 · 중(둘째 자리) = 새 기능 추가 · 소(셋째 자리) = 버그 수정·문구·UI 다듬기. (패널 하단의 v2026.… 는 "항상 최신 실행"을 확인하는 빌드 날짜 표시이고, 아래는 기능 기준 이력입니다.)
